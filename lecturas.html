Conceptos Basicos en el Desarrollo de Software

El Software

El software es la parte lógica o funcional de un sistema informático, es decir, la parte que hace funcionar al conjunto de elementos de origen electrónico de un ordenador, permitiendo y coordinando el funcionamiento de cada uno de los componentes físicos. El software se divide principalmente en dos elementos, que son el software básico o sistema operativo (que trataremos en profundidad en el siguiente punto) y software de aplicación, que es aquel constituido por una serie de programas y datos. El software de aplicación es desarrollado con un objetivo muy concreto, que es el de proporcionar al usuario la posibilidad de realizar una serie de tareas de forma fácil y sencilla con la ayuda del ordenador.

Niveles de abstracción de las partes de un sistema informático.

Se consideran como software de aplicación los programas de comunicaciones, los programas de ofimática (procesadores de texto, hojas de cálculo, bases de datos, programas de presentación gráfica, etc.), programas de gestión (de clientes, facturación, nóminas, etc.), y otros.
